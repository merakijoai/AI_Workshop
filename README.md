# ورشة البرمجة بالذكاء الاصطناعي - Landing Page

A lightweight, responsive one-page website for the AI Programming Workshop event.

## Features

- **Countdown Timer**: Real-time countdown to the workshop (November 29, 2025, 6:00 PM UTC+3)
- **Flipping Minutes**: Minutes display with a smooth flip animation
- **Pulsing Animation**: Timer pulses when 10 seconds remain
- **Conditional Links**: 
  - Shows registration link when more than 1 hour remains
  - Shows attendance link when 1 hour or less remains
- **Gift Unwrapping Animation**: Marketing message appears with ribbon unwrapping effect after timer ends
- **QR Code**: Automatically generated QR code linking to registration form
- **Fully Responsive**: Works on all device sizes

## Setup

1. Add your logo image as `logo.png` in the root directory
2. Open `index.html` in a web browser
3. The countdown will automatically start

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Countdown logic and interactions
- `logo.png` - Your logo (add this file)

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)

## Notes

- The countdown uses UTC+3 timezone
- QR code links to: https://go.meraki-academy.com/RegistrationFormS1AI
- Registration link: https://go.meraki-academy.com/RegistrationFormS1AI
- Attendance link: https://link.meraki-academy.com/AI-Free-Workshop
